# wlctl

RPM packaging for `wlctl`.

- Upstream: <https://github.com/aashish-thapa/wlctl>
- Releases: <https://github.com/aashish-thapa/wlctl/releases>

Notes:

- Rust package with vendored crates generated locally as `Source1`
- Runtime expects `NetworkManager` to be installed and active
- The upstream speed test action is optional and expects `speedtest-cli`
