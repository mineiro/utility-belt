# tflint

RPM packaging directory for `tflint`.

Template: `go`

This package is currently restricted to Fedora rawhide because upstream
requires Go 1.26.2 as of `tflint` 0.62.0.

Upstream:

- Git: https://github.com/terraform-linters/tflint.git
- Releases: https://github.com/terraform-linters/tflint/releases
